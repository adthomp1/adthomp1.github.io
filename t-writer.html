A guide to writing a letter LibreOffice Writer.

Disclaimer

This guide is meant for users of Windows 7, other versions of Windows may not follow the exact process described.

Steps

  1. Open Libre Office using either the desktop icon or the Start menu.

  2. Click "Writer Document" under "Create" in the launch window. (Figure 1)

  3. Type your letter as if it were MS Word.

  4. Once you've written an amazing letter, save it using the "Save" button. (Figure 2)

  5. Choose a good name and where you want to save it. (Figure 3)

  6. Choose a what file format you'd like save it as; Writer offers many options.(Figure 4)

  7. Click "Save" to finish saving your letter.

  8. Exit the program as you would any other.
