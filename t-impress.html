A guide to create a presentation using LibreOffice Impress.

Disclaimer

This guide is meant for users of Windows 7, other versions of Windows may not follow the exact process described.

Steps

  1. Open LibreOffice using either the desktop icon or the Start menu.

  2. Click "Impress Presentation" under "Create" in the launch window. (Figure 1)

  3. Create your slides similar to how you would in MS Power Point.

  4. Once you've made good progress on your presentation, save it using the "Save" button. (Figure 2)

  5. Choose a good name and where you want to save it. (Figure 3)

  6. Choose a what file format you'd like save it as; Impress offers many options.(Figure 4)

  7. Click "Save" to finish saving your presentation.

  8. Exit the program as you would any other.
